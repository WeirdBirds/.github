# Jay Engine

Jay is a collection of Jai modules that together form a game engine.

The most complete modules right now are:
[Jay_Math](https://github.com/WeirdBirds/Jay_Math) - A faster, feature-rich version of the default Math module. Its main downside is that it currently only supports x86 platforms. PRs are welcome.

[Jay_ECS](https://github.com/WeirdBirds/Jay_ECS) - An experimental compile-time-ish ECS. It is not under active development until the renderer is closer to completion.

[Jay_Core](https://github.com/WeirdBirds/Jay_Core) - The main engine module. It is a thin layer of glue that holds [everything together](https://github.com/WeirdBirds/Jay/tree/main).
